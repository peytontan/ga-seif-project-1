User's Board

Enemy's board

There are a total of 5 ships which occupies a 17 grids. The fastest player that manage to sink all 17 grids/5 ships, wins.

Ships are currently randomly placed for both PC and user with the following requirements:
1. There is no overlapping of ships at any point in time
2. Horizontally placed ships will not be broken up into 2 rows
3. Ships will not be placed out of the grids, it will only be placed in the board itself
4. Ships are not diagonally placed, only horiztonally or vertically

User can sink the enemy's ships by randomly selecting a grid at every turn on the right board titled Enemy's board.
User and PC will take turns, user will start first.

User's ships are denoted by the grey grids.
If a ship was hit, the grid will be changed to red.
If a ship was missed, the grid will be changed to green.

The 5 types of ships are:
1. Carrier - 5 grids
2. Battleship - 4 grids
3. Cruiser - 3 grids
4. Submarine - 3 grids
5. Destroyer - 2 grids
